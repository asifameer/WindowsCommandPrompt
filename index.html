    Windows Command Line Interface (CLI) or Prompt

    Using WCLI Terminal

    • Open Windows Command Prompt in Windows 10
      • Right click Windows Startup Icon
      • Click Command Prompt of Command Prompt (Admin)
    • Change window size
      • Write mode <no of characters wide>, <no of lines high> to change the size of the window.
      • For example, mode 100, 50 will make the window 100 characters wide and 50 lines high
    • Close the Command Prompt: Alt + F4
    • Copy Text
      • Click on the icon in the title bar and choose Edit > Mark or Ctrl+M
      • Select the area to be copied and hit Enter
    • Find text
      • Click the icon in the tile bar and choose Edit>Find or Ctrl+F
    • F7 for command line history
      • Hit F7 and you will get a complete list of commands that you executed.
      • Use the arrow keys to highlight the command you want to run again or
      • just hit the number key corresponding to the command that you want to execute.
    • Fullscreen
      • Hit Alt+Enter for full screen
      • Hit Alt+Enter again to go back to previous state
    • History keys
      • Use F1 to paste the previous command one character at a time
      • Press F2 (followed by a number) to paste the previous command up to the said number
      • Press F3 to paste the previous command.
    • Paste Text
      • Click on the icon in the title bar and choose Edit > Paste or Ctrl+V
    • Select All
      • Click the icon in the tile bar and choose Edit>Select All or Ctrl+A

    Common Commands

    • cd changes the directory
    • cd .. to go back to upper level directory
    • del to delete file del example.exe
    • dir - lists the contents of the current directory.
    • driverquery provides lis of all drivers installed on the system
    • E: moves to E: drive.
    • Get help for any command is commandname/? in the WCP
    • mkdir to make new directory mkdir dirname
    • rmdir to remove directory rmdir dirname
    • systeminfo retrieve extensives information about your system, including Windows serial number, computer model, and available RAM
